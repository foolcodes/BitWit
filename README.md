# ⚡ BitWit

> *Your all-in-one hub for Competitive Programming & DSA.*

**BitWit** is a sleek, powerful platform designed to **simplify your coding journey**. It brings together top CP/DSA resources, daily challenges, and upcoming contests — all in one place, with a **visually stunning UI powered by Three.js**.

No more bouncing between tabs. Just focus, code, and grow.

---

## 🚀 Features

- 🧩 **Famous CP/DSA Sheets**: Access top sheets like NeetCode 150, CSES, Love Babbar, and more
- 🔔 **Problem of the Day (POTD)**: Stay sharp with daily challenges from LeetCode, GFG, and others
- 🗓️ **Contest Tracker**: Never miss a coding contest again — all platforms, one calendar
- 🎨 **Immersive UI**: Smooth animations and beautiful 3D experiences using Three.js + Framer Motion
- 🔍 **Search & Filter**: Quickly navigate through sheets and problems

There’s no central place to keep track of:
- All your favorite CP/DSA sheets
- Daily practice problems from different platforms

**BitWit** solves that — with a blend of utility and aesthetics 💻✨
